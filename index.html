Ciência da computação

Portifólio dos projetos realizados por Matheus Zanellato Fagundes da Silva durante o primeiro período de ciência da computação











Primeiro Projeto

Segundo Projeto

    O segundo projeto foi a construção de um aplicativo/jogo utilizando o Processing, que é um ambiente de desenvolvimento integrado, sendo Java a linguagem de programação utilizada. O jogo se baseia em desviar dos quadrados que estão caindo, podendo se movimentar para esquerda e direita.







Terceiro Projeto

     O terceiro projeto foi a construção do jogo batalha naval utilizando o Microsoft Visual Studio, um ambiente de desenvolvimento integrado da Microsoft, criado como aplicativo do Windows Forms em C#. O jogador escolhe as coordenadas de suas embarcações e , após isso, tenta acertar as embarcações inimigas pelas coordenadas.
